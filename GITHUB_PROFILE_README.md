## Master's Thesis

>**[Data Reliability in EEG-based Emotion Recognition: A Framework for Unreliable Data Pair Removal](https://dl.nanet.go.kr/SearchDetailView.do?cn=KDMT12025000014380&sysid=nadl)**  
[Affective AI Lab](https://www.affctiv.ai) | Advisor: Byung Hyung Kim

Proposed a data cleansing framework that identifies and removes unreliable EEG-label pairs — samples where the correlation between EEG signals and emotion labels is degraded — using deep learning-based Out-of-Distribution (OOD) detection. Achieved up to **+6.14% improvement in classification accuracy** and **+4.33% in AUROC** over the conventional method (Riemannian Potato) across three public datasets.
